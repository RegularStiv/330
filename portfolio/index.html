    This game was my first cooperative education experience and I worked on many different areas as needed. I worked as a programmer, tech artist, 3D modeler, and level designer.     Snake

    I made this simple game to make Snake harder as you continue the game by getting faster as you get longer. The reason I made this game was to learn more about how games in JavaScript functioned and Howlr specifically. The game keeps track of your score, high score, and the level you are on currently. The game is run solely in JavaScript HTML and CSS.
