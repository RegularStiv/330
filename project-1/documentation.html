Major Features

The API can sort through various spells based on level and spells from specific classes. It cannot however filter both at the same time so I manually sorted through both using dropdowns. The search bar is not filtered by the drop downs because you are searching for a specific spell. You are able to favorite spells now however for the final project I want to be able to store spells based on a specifc character that you make.
